If you want to make your own game or edit someone else's game made with this engine, you just need to change:
game_objects.py
or
list_of_game_objects.json


To add an object type such as the player or a new type of wall, make a new class in "game_objects.py" and list it in "list_of_game_objects.json".
In the class, you put what you want the object to do each frame it's active in a function called object "logic_frame()"
